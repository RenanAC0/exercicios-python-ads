## Exercícios de Python — ADS

Exercícios que fiz durante o primeiro semestre da faculdade, organizados por tema pra ficar mais fácil de navegar.

### Estrutura

**logica/** — exercícios com condicionais e match/case
- `fizz_buzz.py`
- `placa_carro.py` — calcula o dia de rodízio a partir do final da placa
- `calculadora_parcelamento.py` — calcula juros conforme número de parcelas
- `aprovacao_aluno.py` — verifica aprovação por nota e frequência

**funcoes/** — exercícios praticando funções
- `calculadora_menu.py` — calculadora com menu de opções
- `sistema_hospedagem.py` — calcula valor da diária por tipo de quarto
- `calculadora_dias_semana.py` — calcula em que dia da semana cai um evento futuro

**estruturas/** — exercícios com listas
- `sorteio_copa.py` — sorteia confrontos entre seleções
- `historico_navegador.py` — simula histórico de navegação usando lista como pilha

**conversao/** — conversão de bases numéricas
- `binario_para_decimal.py`
- `decimal_para_binario.py`

**testes/** — meu primeiro contato com testes automatizados
- `eh_par.py` — função simples que verifica se um número é par
- `test_eh_par.py` — teste da função acima, usando pytest

### Rodando os testes

```
pip install pytest
pytest testes/test_eh_par.py
```

Esses exercícios foram passados em aula ou em listas da disciplina de Programação de Computadores.
